backend文件夹：java后端代码（springboot项目）

admin文件夹：管理端（前端vue3）。进入后执行 npm install && npm run dev，浏览器访问 http://localhost:5173

client文件夹：用户端（前端vue3）。进入后执行 npm install && npm run dev，浏览器访问 http://localhost:5174

联调前请先启动后端（backend 目录下 mvn spring-boot:run），接口基地址 http://localhost:8088/api，可与 Apifox 并行测试。

DOCX 目录：`后端接口文档.md`、`前端接口文档.md`、各 `功能文档-*.md`、`功能文档索引.md`、**`测试流程说明.md`**（整体测试步骤与安全/限流说明索引）。
